# 🎮 2D Platformer Game (Unity, C#)

## 📌 Описание проекта

Данный проект представляет собой **2D платформер**, разработанный на **Unity (C#)** и предназначенный для публикации на платформе **Яндекс Игры**.

Игрок управляет персонажем, который перемещается по уровням, преодолевает препятствия, сражается с врагами и собирает ресурсы.

---

## 🚀 Основные возможности

* 🏃‍♂️ Движение персонажа:

  * Влево / вправо
  * Прыжки
  * Приседание
* 🔫 Боевая система:

  * Подбор оружия
  * Стрельба
* ❤️ Система здоровья:

  * Отображение жизней (HP)
  * Получение урона
* 💰 Очки / ресурсы:

  * Сбор предметов
  * Подсчёт очков
* 🌍 Уровни:

  * Несколько уникальных локаций

---

## 🌍 Примеры уровней

1. 🌲 **Дремучий лес**
2. 🏜 **Пустыня**
3. 💧 **Водная долина**

Каждый уровень имеет:

* Уникальный дизайн
* Препятствия
* Врагов
* Атмосферу

## 📦 Сборка проекта

1. Открыть проект в Unity Hub
2. Выбрать платформу 
3. Нажать **Build**
4. Загрузить билд на платформу Яндекс Игры
